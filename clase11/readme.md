# Clase 11 – JavaScript

Ejercicio 1

Programa que solicita un numero al usuario y determina si es positivo negativo o cero

Ejercicio 2

Semaforo inteligente donde el usuario ingresa un color y el programa indica la accion correspondiente

Ejercicio 3

Conteo desde 1 hasta un numero ingresado por el usuario utilizando un ciclo for

Ejercicio 4

Se ingresa un numero y se determina si es par o impar

Ejercicio 5

Se muestran los numeros pares entre dos valores ingresados por el usuario validando los datos

Ejercicio 6

Tabla de multiplicar de un numero del 1 al 10

Ejercicio 7

Calculadora simple con suma resta multiplicacion y division utilizando switch

Ejercicio 8

Se muestran los multiplos de un numero entre 1 y 100

Ejercicio 9

Cuenta regresiva desde 10 hasta 0 y mensaje final Despegue

Ejercicio 10

Juego para adivinar un numero aleatorio entre 1 y 10 con hasta 3 intentos

Observaciones

### Ejercicio 10
La computadora genera un número aleatorio entre 1 y 10 y el usuario tiene hasta 3 intentos para adivinarlo.

## Observaciones
Los ejercicios se ejecutan desde la terminal utilizando Node.js y la librería `prompt-sync`.
