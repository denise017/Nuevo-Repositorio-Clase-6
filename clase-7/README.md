
##Actividad Clase 7 – Duelo de lenguajes
---

1) Sintaxis:
Al escribir el codigo se nota que c es mas estructurado y tiene mas pasos para hacer lo mismo para mostrar “Hola Mundo” necesita varias lineas y una estructura fija
En javascript con una sola linea se logra el mismo resultado lo que lo hizo mas simple
---

2) Archivo guardado:
El archivo .c no se puede ejecutar directamente primero necesita ser compilado con un compilador y javascript se puede ejecutar directamente con su entorno
---

3) Ejecucion:
En javascript los errores se detectan en el momento en que el programa intenta leer la linea con error
---

4) Observacion:
El codigo en c no es tan facil de leer al principio algunas palabras como #include int o return no son tan intuitivas y necesitan explicacion previa

-include se usa para incluir librerias que el programa necesita para funcionar
-int indica el tipo de dato que devuelve la funcion main
-return se usa para finalizar el programa y devolver un valor
